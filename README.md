# NewCool Civil - Derecho Civil Chile

Portal educativo sobre derecho civil y herramientas juridicas.

## Demo

[https://civil.newcool.io](https://civil.newcool.io)

## Caracteristicas

- Calculadora de intereses legales
- Calculadora de plazos procesales
- Calculadora de costas judiciales
- Guia de juicios civiles
- Informacion de procedimientos

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
